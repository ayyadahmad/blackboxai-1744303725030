Welcome to CodeCraft!

Your all-in-one toolkit for web development. Extract, analyze, and reuse code from any website with ease.

Quick Start Guide

1. Click the Extension

Click the CodeCraft icon in your browser's toolbar to open the main interface.

2. Select Elements

Use the inspector tool to select elements on any webpage.

3. Use Tools

Extract code, copy styles, or export elements with a single click.

Key Features

Element Cloning

Clone any element from a website with a single click. Perfect for reusing components.

Code Extraction

Extract HTML, CSS, and JavaScript code from any website element.

Color Selection

Pick colors from any website and copy their values instantly.

Website Crawler

Crawl entire websites to extract and analyze code patterns.

Code Comparison

Compare code snippets from different sources side by side.

Export Options

Export code to CodePen, download as files, or copy to clipboard.

Keyboard Shortcuts

Toggle Inspector Ctrl + Shift + I
Copy Element Ctrl + Shift + C
Color Picker Ctrl + Shift + P
Quick Export Ctrl + Shift + E
